# React sample

Integración mínima para React usando un `useEffect` client-side.

## Qué muestra

- render básico con React
- uso de CSS vars en `:root`
- import dinámico dev-only del overlay browser dentro de un efecto
- cleanup correcto del handle en el retorno de `useEffect`

## Archivos clave

- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/styles.css`
