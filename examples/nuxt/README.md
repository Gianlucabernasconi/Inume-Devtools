# Nuxt sample

Integración mínima para Nuxt usando un plugin `.client` solo en desarrollo.

## Qué muestra

- plugin client-only
- guard dev-only en runtime
- patrón recomendado de import dinámico para el entry `browser`

## Archivos clave

- `app.vue`
- `plugins/css-vars-devtools.client.ts`
- `assets/css/main.css`
